BitStore

Gotta bit? We can store it!

Our fully scaleable bit store API is fully REST complient. Everything you need to store a bit in one simple API.

POST /bit

Set the value of your bit. 

      key:   <your API key>
      value: anything that can true or false, e.g. t, f, 0, 1, etc.

GET /bit?key=<your API key>

Return the value of your bit.
